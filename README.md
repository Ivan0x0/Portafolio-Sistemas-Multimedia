## Descripción del Proyecto
Este repositorio contiene el portafolio final que integra múltiples disciplinas y tecnologías desarrolladas durante la gestión académica. El sistema funciona sobre un servidor web local y actúa como un menú centralizado para evaluar lógica de negocios, procesamiento digital de imágenes, renderizado 3D y simulaciones en tiempo real.

## Requisitos Previos del Sistema
Para la correcta evaluación de todos los módulos, el entorno local del evaluador 
debe contar con el siguiente software instalado:

1. **XAMPP (o similar):** Para la ejecución del servidor Apache y procesamiento de código PHP.
2. **Python 3.x:** Requerido para los algoritmos de Visión Artificial.
